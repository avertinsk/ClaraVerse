# Директория функций

Эта директория содержит модули, основанные на функциях. Каждая функция должна быть самодостаточной с собственными компонентами, хуками и логикой.

## Пример структуры

```
features/
├── auth/
│   ├── components/
│   ├── hooks/
│   ├── services/
│   └── index.ts
└── dashboard/
    ├── components/
    ├── hooks/
    └── index.ts
```

## Рекомендации

- Каждая функция должна экспортировать свой публичный API через файл index.ts
- Держите логику, специфичную для функции, изолированной в директории функции
- Общие утилиты размещайте в директории utils верхнего уровня
