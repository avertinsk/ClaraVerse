# Веб-сайт

Этот веб-сайт создан с использованием [Docusaurus](https://docusaurus.io/) — современного генератора статических веб-сайтов.

## Установка

```bash
yarn
```

## Локальная разработка

```bash
yarn start
```

Эта команда запускает локальный сервер разработки и открывает окно браузера. Большинство изменений отображаются в реальном времени без необходимости перезапуска сервера.

## Сборка

```bash
yarn build
```

Эта команда генерирует статический контент в директорию `build` и может обслуживаться любым сервисом хостинга статического контента.

## Развёртывание

С использованием SSH:

```bash
USE_SSH=true yarn deploy
```

Без использования SSH:

```bash
GIT_USER=<Your GitHub username> yarn deploy
```

Если вы используете GitHub Pages для хостинга, эта команда — удобный способ собрать веб-сайт и отправить в ветку `gh-pages`.
