# Настройка Anthropic

Подключите ClaraVerse к Anthropic для использования моделей Claude для бесед, анализа и генерации кода.

## Получение API-ключа

1. Перейдите на [console.anthropic.com](https://console.anthropic.com/) и войдите (или создайте аккаунт).
2. Перейдите в **API Keys** на панели управления.
3. Нажмите **Create Key**, назовите его и сразу скопируйте ключ — он показывается только один раз.
4. Добавьте способ оплаты в разделе **Billing**, если ещё не сделали. Доступ к API требует активного тарифного плана.

## Добавление Anthropic в ClaraVerse

1. Откройте **Панель администратора > Провайдеры**.
2. Нажмите **Добавить провайдер**.
3. Введите имя (например, «Anthropic»).
4. Установите базовый URL `https://api.anthropic.com`.
5. Вставьте ваш API-ключ.
6. Нажмите **Сохранить**.

Модели Anthropic появятся во всех селекторах моделей после сохранения провайдера.

## Доступные модели

| Модель | Лучше всего для |
|-------|----------|
| **Claude Opus 4** | Сложного анализа, нюансированного письма, глубоких рассуждений |
| **Claude Sonnet 4** | Сбалансированной производительности и скорости для большинства задач |
| **Claude Haiku** | Быстрых, лёгких ответов, высокой пропускной способности |

Все модели Claude поддерживают большие окна контекста (до 200K токенов) и работают с текстовыми, кодовыми и визуальными входными данными.

## Советы

- **Начните с Sonnet.** Claude Sonnet 4 — лучший универсальный выбор для большинства рабочих нагрузок — быстрый, способный и экономичный.
- **Используйте Opus для сложных задач.** Оставьте Claude Opus 4 для задач, требующих глубоких рассуждений, сложной генерации кода или долгосрочного анализа.
- **Haiku для скорости.** Когда задержка важнее глубины (быстрая классификация, короткие ответы, маршрутизация), Haiku — самый быстрый вариант.
- **Биллинг workspace.** Если вы являетесь частью workspace Anthropic, убедитесь, что API-ключ принадлежит правильному workspace, чтобы биллинг был правильно атрибутирован.
- **Лимиты скорости.** Anthropic применяет лимиты скорости для каждой модели. Если вы видите ошибки 429, проверьте лимиты вашего уровня на [console.anthropic.com](https://console.anthropic.com/) и рассмотрите запрос на увеличение лимита.
