# Поддержка нескольких провайдеров

ClaraVerse подключается к нескольким ИИ-провайдерам одновременно. Вы приносите свои API-ключи (BYOK), выбираете нужные модели и свободно переключаетесь между ними в Чате, Рабочих процессах, Nexus и Routine.

## Как это работает

1. **Администратор настраивает провайдеров** — Администратор добавляет один или несколько ИИ-провайдеров в Панели администратора и опционально устанавливает API-ключ по умолчанию для каждого.
2. **Пользователи добавляют свои ключи** — Отдельные пользователи могут добавить личные API-ключи в **Настройки > Провайдеры**. Личные ключи переопределяют ключи, предоставленные администратором.
3. **Модели появляются автоматически** — После подключения провайдера его модели отображаются во всех селекторах моделей на платформе.

## Поддерживаемые провайдеры

| Провайдер | Тип | Требуется API-ключ | Примечания |
|----------|------|-----------------|-------|
| [OpenAI](./openai.md) | Облако | Да | GPT-4o, o1, o3 и другие |
| [Anthropic](./anthropic.md) | Облако | Да | Claude Opus 4, Sonnet 4, Haiku |
| [Google AI](./google.md) | Облако | Да | Gemini 2.5 Pro, Flash |
| [Ollama](./ollama.md) | Локальный | Нет | Бесплатно, полностью офлайн |
| [Пользовательский / OpenAI-совместимый](./custom.md) | Различный | Зависит | LM Studio, vLLM, Groq, Together AI и т.д. |

## Ключи администратора vs ключи пользователей

- **Ключ администратора** — Устанавливается в Панели администратора. Используется всеми пользователями, у которых нет своего ключа для этого провайдера.
- **Ключ пользователя** — Устанавливается в настройках пользователя. Имеет приоритет над ключом администратора. Только запросы этого пользователя используют его.

Если ключ не настроен ни на одном уровне, провайдер виден, но запросы завершатся ошибкой, пока ключ не будет добавлен.

## Добавление провайдера (быстрые шаги)

1. Перейдите в **Панель администратора > Провайдеры**.
2. Нажмите **Добавить провайдер**.
3. Введите имя (например, «OpenAI»), базовый URL и ваш API-ключ.
4. Сохраните. Модели этого провайдера теперь доступны всем пользователям.

См. отдельные страницы провайдеров для подробных инструкций по настройке.
