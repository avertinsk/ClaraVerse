# ИИ-документы

ClaraVerse включает встроенные инструменты для обработки документов с помощью ИИ. Загружайте PDF, презентации PowerPoint, файлы CSV и другие типы документов, затем анализируйте, суммируйте, преобразовывайте или извлекайте данные из них прямо в чате.

## Поддерживаемые типы документов

| Формат | Что можно делать |
|--------|----------------|
| **PDF** | Суммировать, извлекать текст, отвечать на вопросы по содержанию, извлекать таблицы |
| **PowerPoint (PPT/PPTX)** | Анализировать слайды, извлекать ключевые моменты, генерировать заметки докладчика, реструктурировать контент |
| **CSV** | Анализировать данные, генерировать статистику, фильтровать строки, создавать графики, находить паттерны |
| **Другие форматы** | Текстовые файлы, файлы кода и другие распространённые форматы также можно загружать и обрабатывать |

## Как использовать

1. **Загрузите** документ в чат, нажав кнопку вложения или перетащив файл в беседу.
2. **Спросите Clarу**, что вы хотите с ним сделать.

**Примеры промптов:**

- «Суммируй этот PDF в 5 пунктах»
- «Каковы ключевые выводы из этой презентации?»
- «Проанализируй этот CSV и скажи, у какого продукта был самый высокий доход»
- «Извлеки все адреса электронной почты из этого документа»
- «Создай график по столбцу B против столбца D в этой таблице»

## Ключевые возможности

- **Суммаризация** — Получайте краткие резюме длинных документов.
- **Вопросы и ответы** — Задавайте конкретные вопросы и получайте ответы, основанные на содержимом документа.
- **Анализ данных** — Загружайте CSV или табличные данные и получайте статистические данные, тенденции и визуализации.
- **Извлечение контента** — Извлекайте конкретную информацию, такую как таблицы, имена, даты или цифры.
- **Трансформация** — Преобразовывайте содержимое документа в разные форматы (пункты, таблицы, структурированные данные).

## Советы

- Для больших PDF попросите Clarу сосредоточиться на конкретных разделах или диапазонах страниц для более точных ответов.
- При работе с данными CSV укажите имена столбцов или опишите данные, чтобы Clara могла быстро ориентироваться.
- Вы можете загрузить несколько документов в одну беседу и попросить Clarу сравнить или сопоставить их.
- ИИ-документы работают с любым провайдером моделей (OpenAI, Anthropic, Ollama и т.д.) — просто убедитесь, что модель выбрана в вашем чате.
