# 150+ встроенных интеграций

![Интеграции](/img/features/integrations.png)

ClaraVerse поставляется с 150+ интеграциями из коробки. Подключайтесь к сервисам, которые вы уже используете — без MCP-серверов, без плагинов, без дополнительной настройки, кроме ввода учётных данных.

## Доступные интеграции

Частичный список поддерживаемых сервисов:

| Категория | Сервисы |
|---|---|
| **Коммуникация** | Slack, Discord, Telegram, Microsoft Teams |
| **Управление проектами** | Jira, Linear, Asana, Trello, Notion |
| **Разработка** | GitHub, GitLab, Bitbucket |
| **Продуктивность** | Google Sheets, Google Drive, Google Calendar, Airtable |
| **CRM и маркетинг** | HubSpot, Salesforce, Mailchimp |
| **Электронная коммерция** | Shopify, Stripe |
| **Аналитика** | Mixpanel, PostHog, Google Analytics |
| **Облако и инфраструктура** | AWS S3, Google Cloud Storage |
| **Базы данных** | MongoDB, Redis, PostgreSQL, MySQL |
| **Другое** | Twilio, SendGrid, Zapier Webhooks, пользовательский HTTP |

Это не полный список. Проверьте **Настройки > Интеграции** в приложении для полного списка.

## Настройка учётных данных

1. Перейдите в **Настройки > Интеграции**.
2. Найдите сервис, к которому хотите подключиться.
3. Введите необходимые учётные данные (API-ключ, OAuth-токен или строка подключения в зависимости от сервиса).
4. Нажмите **Сохранить**.

Вот и всё. Интеграция теперь доступна во всех функциях ClaraVerse.

## Общие для всего

После настройки интеграции доступны в:

- **Чат** — Clara может получать данные из подключённых сервисов прямо во время беседы.
- **Рабочие процессы** — используйте узлы интеграций в ваших визуальных автоматизациях.
- **Nexus** — Clara получает доступ к интеграциям при работе над долгосрочными задачами.
- **Routine** — запланированные задачи могут читать и записывать в любой подключённый сервис.

Подключите один раз, используйте везде.

## MCP не требуется

Эти интеграции **встроены**. Вам не нужно запускать MCP-серверы или устанавливать внешние инструменты для их использования. Они работают сразу после добавления учётных данных.

Если вы хотите использовать MCP для дополнительных инструментов или доступа к локальной файловой системе, ClaraVerse поддерживает и это через [Clara Companion](./companion.md) — но это полностью необязательно.

## Советы

- **Принцип наименьших привилегий.** При создании API-ключей для интеграций ограничивайте их минимальными разрешениями, необходимыми Clare.
- **Тестируйте после подключения.** Откройте Чат и попросите Clarу что-то, использующее интеграцию (например, «Покажи мои открытые задачи GitHub»), чтобы проверить, что подключение работает.
- **Несколько аккаунтов.** Вы можете подключить несколько аккаунтов для одного сервиса, если нужно (например, две разные организации GitHub).
