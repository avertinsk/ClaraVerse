# ИИ-чат с навыками

![Чат](/img/features/chat-demo.gif)

Clara — ваш ИИ-ассистент. Задавайте вопросы, получайте ответы и позволяйте Clare использовать **навыки** — контекстно-зависимые инструменты, которые активируются в ходе беседы, когда это необходимо. Без ручного переключения.

## Как это работает

1. Откройте **Чат** на боковой панели.
2. Выберите модель (или оставьте значение по умолчанию).
3. Введите сообщение. Clara отвечает и автоматически активирует навыки, когда они релевантны.

Например, спросите «Что произошло в новостях сегодня?» и Clara самостоятельно запустит **веб-поиск**. Попросите «Создай логотип для моего проекта» и включится **генерация изображений**.

## Навыки

![Навыки](/img/features/skills.png)

Навыки — это инструменты, которые Clara может вызывать во время общения с вами. Они включают:

- **Веб-поиск** — актуальные результаты через встроенный экземпляр SearXNG.
- **Генерация изображений** — создание изображений по текстовым описаниям.
- **Анализ данных** — обработка CSV, выполнение расчётов, создание графиков.
- **Выполнение кода** — написание и запуск фрагментов кода прямо в беседе.
- **Инструменты для документов** — создание PDF, PowerPoint и многого другого.

Навыки выбираются ИИ на основе контекста. Вам не нужно включать их вручную.

## Поддержка нескольких провайдеров

Используйте любого поддерживаемого провайдера моделей:

| Провайдер | Примеры |
|---|---|
| **OpenAI** | GPT-4o, GPT-4.1 |
| **Anthropic** | Claude Opus, Sonnet |
| **Google** | Gemini Pro, Flash |
| **Ollama** | Llama, Mistral, Qwen (автоопределение) |
| **Любой OpenAI-совместимый endpoint** | LM Studio, vLLM и т.д. |

Переключайте провайдеров для каждой беседы через селектор моделей.

## Человек в цикле

Clara задаёт **визуальные уточняющие вопросы**, когда ей нужен ваш ввод. Вместо текстовых подсказок вы получаете типизированные формы — выпадающие списки, флажки, селекторы изображений — чтобы вы могли отвечать точно, не угадывая, какой формат ожидает ИИ.

## Хранение бесед

Все беседы хранятся **локально в вашем браузере** с использованием IndexedDB. Ничего не покидает ваш компьютер, если вы не отправите сообщение облачному провайдеру. Это архитектура с нулевым знанием: сервер никогда не хранит историю ваших чатов.

## Советы

- **Используйте интеграции в чате.** Clara может получать данные из Slack, GitHub, Jira и 150+ других сервисов прямо в беседе. Настройте учётные данные один раз в Настройки > Интеграции.
- **Артефакты.** Код, графики и сгенерированный контент появляются как интерактивные артефакты, к которым можно вернуться через панель Артефактов.
- **Память.** Clara запоминает контекст между беседами и автоматически архивирует старые воспоминания, оставаясь полезной с течением времени.
