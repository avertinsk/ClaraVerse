# Устройства

Подключайте несколько машин к вашему экземпляру ClaraVerse. Каждое устройство, запускающее [Clara Companion](./companion.md), регистрирует себя и предоставляет свои локальные MCP-серверы для Clare.

## Как это работает

1. Установите [Clara Companion](./companion.md) на любую машину
2. Выполните `mcp-client login` для аутентификации с вашим экземпляром ClaraVerse
3. Устройство появится на вкладке **Настройки > Устройства**
4. Clara теперь может удалённо использовать MCP-инструменты с этого устройства

## Управление устройствами

Откройте **Настройки** и перейдите на вкладку **Устройства**. Отсюда вы можете:

- **Просматривать подключённые устройства** — видеть имя устройства, платформу, версию, время последней активности и местоположение
- **Переименовывать устройство** — дать ему понятное имя для лёгкой идентификации
- **Отзывать устройство** — отключить и удалить устройство из вашего аккаунта

Вы также можете управлять устройствами из CLI:

```bash
mcp-client devices list
mcp-client devices rename <device-id> "My Laptop"
mcp-client devices revoke <device-id>
```

## Варианты использования

- **Работа + Дом** — подключите обе машины, чтобы Clara имела доступ к файлам на любой из них
- **Доступ к серверу** — запустите Clara Companion на удалённом сервере, чтобы дать Clare доступ к серверным инструментам
- **Командные настройки** — каждый член команды подключает своё устройство со своими MCP-серверами

## Советы

- Каждое устройство поддерживает свой набор MCP-серверов, настроенных в `~/.claraverse/mcp-config.yaml`
- Устройства автоматически переподключаются при обрыве соединения
- Отзыв устройства немедленно отключает его и удаляет его инструменты у Clare
- Для продакшен-развёртываний используйте обратный прокси с TLS, чтобы связь между устройствами была зашифрована через WSS
