# Nexus — ваша доска Trello, но ИИ — ваша команда

![Nexus](/img/features/nexus.png)

Думайте о Nexus как о Trello — но вместо назначения карточек людям вы назначаете их ИИ. Clara — ваша команда. Размещайте задачи на доске, и она берёт их, выполняет шаг за шагом и перемещает по колонкам по мере продвижения. Исследования, программирование, анализ данных, написание контента — делегируйте всё и наблюдайте, как доска обновляется в реальном времени.

## Как это работает

1. Откройте **Nexus** на боковой панели.
2. Введите задачу в поле ввода внизу представления Nexus. Примеры:
   - «Исследуй топ-5 конкурентов в сфере управления проектами и суммируй их ценообразование.»
   - «Рефакторинг модуля аутентификации для использования JWT refresh-токенов.»
   - «Проанализируй CSV продаж за прошлый квартал и выдели тенденции.»
3. Clara берёт задачу, выполняет её шаг за шагом и перемещает по доске по мере продвижения.

Вы назначаете работу. Clara её выполняет. Вы проверяете результаты.

## Канбан-доска

Задачи проходят через колонки, как на любой проектной доске:

| Колонка | Значение |
|---|---|
| **Черновик** | Задача создана, но ещё не отправлена |
| **В очереди** | Отправлена и ожидает, когда её возьмут |
| **В работе** | Clara активно работает над ней |
| **Готово** | Завершена и готова к вашей проверке |
| **Ошибка** | Задача столкнулась с ошибкой и не могла быть завершена |

Нажмите на любую карточку задачи, чтобы увидеть полный журнал выполнения — каждый шаг, который сделала Clara, каждый вызванный инструмент, каждое принятое решение. Полная прозрачность, без чёрного ящика.

## Проекты

Группируйте связанные задачи в **Проекты**, чтобы организовать вашу доску. Проектом может быть «Маркетинговое исследование Q1» или «Рефакторинг backend». Фильтруйте доску по проекту, чтобы сосредоточиться на важном.

## Что делает его мощным

- **Делегируйте реальную работу.** Это не быстрые ответы в чате. Nexus обрабатывает задачи, занимающие минуты или часы — исследовательские отчёты, рефакторинг кода, анализ данных.
- **Полная прозрачность.** Каждое действие Clara логируется. Вы видите точно, что она сделала и почему.
- **150+ интеграций.** Clara может получать задачи GitHub, читать Google Sheets, публиковать в Slack — всё это работая над вашими задачами.
- **Многошаговое мышление.** Clara разбивает сложные запросы на подзадачи и выполняет их последовательно, как это сделал бы член команды.

## Советы

- **Будьте конкретны.** «Проанализируй данные продаж» — нормально. «Проанализируй продажи Q4 из прикреплённого CSV, сравни помесячный рост и отметь любой продукт с падающим доходом» — лучше.
- **Следите за прогрессом.** Нажмите на выполняющуюся задачу, чтобы увидеть обновления в реальном времени и промежуточные результаты.
- **Комбинируйте с Routine.** Планируйте повторяющиеся задачи с помощью Routine, чтобы они выполнялись автоматически по расписанию.
