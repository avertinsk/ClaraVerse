# Что такое ClaraVerse?

ClaraVerse — это self-hosted приватное ИИ-пространство. Вы запускаете его на своей собственной инфраструктуре, подключаете свои ИИ-провайдеры (OpenAI, Anthropic, Google, Ollama или любую OpenAI-совместимую конечную точку) и получаете полнофункциональную ИИ-платформу без отправки данных третьим лицам.

Построенный на React-фронтенде, Go-бэкенде и развёртывании Docker Compose, ClaraVerse рассчитан на запуск за считанные минуты.

## Ключевые функции

- **Чат** — Мульти-модельные разговоры с загрузкой файлов, выполнением кода и встроенным веб-поиском.
- **Навыки** — 150+ встроенных инструментов, активирующихся во время разговора — веб-поиск, генерация изображений, анализ данных и многое другое.
- **Рабочие процессы** — Визуальный конструктор автоматизации с перетаскиванием для цепочки ИИ-задач.
- **Nexus** — Поручайте Кларе долгосрочные задачи (исследование, программирование, анализ). Отслеживайте прогресс на Канбан-доске.
- **Расписания** — Запланированные ИИ-задачи, выполняющиеся автоматически (в стиле cron) и отправляющие отчёты через Telegram.
- **Каналы** — Интеграция с Telegram — общайтесь с Кларой с телефона, получайте отчёты расписаний.
- **Интеграции** — 150+ встроенных интеграций: Slack, GitHub, Jira, Google Sheets, Notion, Discord, Telegram, HubSpot, Shopify и другие. MCP не требуется.
- **Артефакты** — Сгенерированный ИИ интерактивный контент (графики, приложения, визуализации), рендерящийся встроенно.
- **ИИ-документы** — Встроенные инструменты для обработки документов PPT, PDF и CSV.
- **Память** — Клара помнит контекст между разговорами и автоматически архивирует старые воспоминания.
- **Устройства** — Подключайте все ваши машины — Клара удалённо использует MCP-серверы на любом из них.
- **Clara Companion** — Мост локальных MCP-серверов к ClaraVerse через WebSocket.
- **Локальный ИИ** — Ollama и LM Studio обнаруживаются автоматически. Запускайте модели локально без настройки.
- **Хранение в браузере** — Переписки хранятся в IndexedDB. Архитектура с нулевым знанием.
- **BYOK** — Используйте свои ключи или бесплатные локальные модели.
- **Панель администратора** — Управление пользователями, конфигурация провайдеров, управление моделями и аналитика использования.
- **Self-Hosted** — Ваши данные остаются на вашем оборудовании. Лицензия AGPL-3.0.

## Архитектура вкратце

| Компонент | Технология |
|-----------|------------|
| Фронтенд | React 19 + TypeScript + Vite |
| Бэкенд | Go (Fiber) |
| Базы данных | MongoDB (переписки, данные), MySQL (провайдеры, модели), Redis (задачи, pub/sub) |
| Поиск | SearXNG (self-hosted, приватный) |
| Развёртывание | Docker Compose |
