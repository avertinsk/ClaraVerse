# Обзор панели администратора

Панель администратора даёт администраторам экземпляра полный контроль над пользователями, ИИ-провайдерами, моделями и аналитикой платформы. Доступ по адресу `/admin` в вашем развёртывании ClaraVerse.

## Как стать администратором

Первый зарегистрированный пользователь автоматически становится администратором. Учётной записи администратора по умолчанию нет.

### Добавление дополнительных администраторов

**Вариант 1 — Установка роли в MongoDB (рекомендуется):**

```javascript
db.users.updateOne(
  { email: "admin@example.com" },
  { $set: { role: "admin" } }
)
```

**Вариант 2 — Переменная окружения:**

```bash
# .env
SUPERADMIN_USER_IDS=user-id-1,user-id-2
```

## Доступ к панели администратора

| Окружение | URL |
|-------------|-----|
| Разработка | `http://localhost:5173/admin` |
| Продакшен  | `https://yourdomain.com/admin` |

Только пользователи с ролью `admin` или чьи ID указаны в `SUPERADMIN_USER_IDS` могут получить доступ к панели.

## Панель управления

Панель администратора отображает общесистемную статистику в одном месте:

- Всего пользователей
- Активных чатов
- Всего сообщений
- API-вызовов сегодня
- Активных провайдеров
- Всего моделей

Вы можете программно проверить свой статус администратора:

```http
GET /api/admin/me
Authorization: Bearer <access_token>
```

Возвращает `"is_admin": true`, если аутентифицирован как администратор.
